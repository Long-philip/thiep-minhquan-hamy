# Sổ theo dõi khách

| Khách (chú rể & cô dâu) | Repo | Vercel URL | Ngày tạo | Ngày cưới | Ngày hết hạn (khóa) | Trạng thái |
|---|---|---|---|---|---|---|
| _ví dụ:_ Tuấn & Linh | wedding-tuanlinh | tuanlinh-wedding.vercel.app | 2026-07-01 | 2026-07-20 | 2026-07-24 | đang chạy |
|  |  |  |  |  |  |  |

> Trạng thái: `đang sửa` / `đã gửi khách` / `đang chạy` / `đã khóa`.
> Quy ước tên repo: `wedding-<tênchúrể><têncôdâu>` (không dấu, viết liền).
